# NeuroMentor API

## Описание проекта

NeuroMentor API — это серверная часть интеллектуальной системы наставничества, построенная на базе FastAPI и интегрированная с нейросетевыми моделями DeepSeek. Система предназначена для предоставления персонализированных советов и поддержки пользователей, учитывая их возраст, пол и историю предыдущих диалогов.

Проект поддерживает сохранение контекста общения в реляционной базе данных, что позволяет ИИ-наставнику "помнить" предыдущие сессии и выстраивать долгосрочную стратегию взаимодействия с пользователем.

## Основной стек технологий

- **Backend**: Python 3.13 (FastAPI)
- **Database**: PostgreSQL 17 (SQLAlchemy ORM)
- **AI Integration**: OpenAI SDK (адаптировано под DeepSeek API)
- **Containerization**: Docker, Docker Compose

## Предварительные требования

- Установленный Docker и Docker Compose.
- API ключ от DeepSeek.

## Шаги по развертыванию

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/your-repo/NeuroMentor-API.git
   ```

2. **Создайте файл `.env` на основе примера:**
   
   Заполните переменные окружения в `.env`, уделив особое внимание:
   - `DEEPSEEK_API_KEY`
   - `POSTGRES_PASSWORD`
   - портам сервисов

3. **Запустите проект с помощью Docker Compose:**
   ```bash
   docker compose up --build
   ```

После запуска API будет доступно по адресу `http://localhost:8000`. Интерактивная документация Swagger доступна по адресу `http://localhost:8000/api/docs`.

## Основные эндпоинты API (v1)

- **POST `/api/v1/user`**: Регистрация нового пользователя (имя, пол, возраст).
- **POST `/api/v1/chat`**: Отправка сообщения ИИ-наставнику. Поддерживает автоматическое создание или подбор существующей сессии.

## Тестирование

Для проверки работы системы в консольном режиме предусмотрен скрипт `console_chat.py`. Он позволяет пройти цикл регистрации и начать полноценный диалог с ИИ непосредственно из терминала.

```bash
python console_chat.py
```
